# RMF Alignment

## Controls Addressed
- AC-2 → Account management (user role)
- IA-5 → Password policy enforcement (STIG role)
- AU-2 → Audit logging (auditd enabled)
- CM-6 → Configuration baseline (gold image playbook)

## Purpose
Provides a repeatable automated method for building hardened Linux gold images aligned with RMF control families.
